Interface

The main CMS interface. From here you have access to all the elements that modifiable. The top navigation bar has quick acces to Contents (screenshot above), Workflow, Media and a quick create button.

The Workflow page reflects the current state of the posts. When creating a new post it should be saved as a Draft. After that the state of the post can be changed to In Review or Ready. If the post is Ready the publish button will be active (in the workflow interface and also in the editing interface).

The Media page has access to all available media. There are options as upload, download. When looking for specific files it is encouraged to use the provided search.
